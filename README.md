# 🎓 JIT File-Sharing Portal: Academic Resource Hub

**JIT File-Sharing Portal** is a dedicated web platform built specifically for the students and faculty of **Jyothy Institute of Technology (JIT), Bangalore**. It simplifies the exchange of academic materials, from VTU-aligned notes to department-specific lab manuals.

🚀 **[Access the JIT Portal Live](https://sites.google.com/view/jyothifiles/home)**

## ✨ Campus-Specific Features
* **Department Filtering:** Organize files by CSE, ISE, ECE, and other JIT departments.
* **VTU Syllabus Alignment:** Optimized for the latest Visvesvaraya Technological University (VTU) curriculum.
* **Instant Peer Sharing:** Upload and share lab records or previous year question papers in seconds.
* **Mobile-Responsive:** Access notes directly from your phone during study sessions in the JIT library.

## 🛠 Tech Stack
* **Frontend:** React.js / HTML5 / CSS3 (Semantic UI)
* **Deployment:** Vercel
* **Optimization:** SEO-first architecture for local campus discovery.

## 📍 Purpose
This project was developed to solve the "resource fragmentation" issue within **Jyothy Institute of Technology**. Instead of hunting through WhatsApp groups, students can find everything in one verified location.

## 🤝 Contributing
If you are a student at **JIT**, feel free to fork this repo and add features like "Internal Marks Calculator" or "Class Schedule Tracker."
